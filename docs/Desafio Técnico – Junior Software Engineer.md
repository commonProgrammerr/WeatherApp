## **Desafio Técnico – Android** 

Neste desafio, você trabalhará a partir do projeto disponibilizado no repositório: https://github.com/karinegmg/WeatherApp 

O objetivo é avaliar suas habilidades com **Kotlin, Jetpack Compose, arquitetura e análise de código existente** . 

## **Objetivo** 

Evoluir o projeto atual implementando melhorias e novas funcionalidades, garantindo qualidade de código e boa experiência de usuário. 

## **Requisitos** 

## 1. **Tela principal em Compose** 

- Implementar/recriar a tela principal utilizando **Jetpack Compose** . 

- A tela deve apresentar as informações principais do app de forma clara e funcional. 

- Fique à vontade para melhorar o layout, organização e experiência do usuário. 

## 2. **Nova tela (obrigatoriamente em Kotlin + Compose)** 

   - Criar uma nova tela no aplicativo. 

   - O conteúdo da tela é livre (exemplos: detalhes do clima, previsões futuras, configurações, etc.). 

   - A navegação entre as telas deve estar funcional. 

3. **Correção de erros** 

   - Identificar e corrigir os problemas já listados no **README do projeto** 

   - Caso encontre outros problemas relevantes, sinta-se à vontade para corrigi-los também. 

## **Entrega** 

- Disponibilizar o código em um repositório (fork do projeto ou repositório próprio). 

- Disponibilizar um vídeo explicando as mudanças feitas no código 

- Incluir um README com a descrição das alterações realizadas 

## **Diferenciais (não obrigatório)** 

- Tratamento de estados e erros 

- Testes (unitários ou instrumentados) 

- Melhorias visuais e responsividade 

- Histórico de commits organizado, e de fácil navegação
